# Python Programming Foundations (Week 1)

This directory contains Python fundamentals coursework completed as part of the **School of AI** curriculum.  
The focus is on building a strong programming base required for AI, ML, and data-driven systems.

---

## Course Structure

### Day 1: Introduction to Python & Development Setup
- Python installation and environment setup
- Running scripts and basic syntax
- Understanding interpreters and editors

### Day 2: Control Flow in Python
- Conditional statements (`if`, `elif`, `else`)
- Loops (`for`, `while`)
- Control keywords (`break`, `continue`, `pass`)

### Day 3: Functions and Modules
- Defining and calling functions
- Parameters, return values
- Importing and using modules
- Code reusability principles

### Day 4: Data Structures
- Lists
- Tuples
- Dictionaries
- Sets
- Iteration and common operations

### Day 5: Working with Strings
- String indexing and slicing
- Common string methods
- Formatting techniques

### Day 6: File Handling
- Reading from files
- Writing to files
- File modes and context managers

### Day 7: Pythonic Code & Mini Project
- Writing clean, readable Python
- Best practices and conventions
- Applying concepts in small project work

---

## Folder Contents

Each file or notebook in this directory corresponds to a specific topic or day of the coursework.

**Naming convention:**
- `day_<number>_<topic>.py` or `.ipynb`
- One concept per file
